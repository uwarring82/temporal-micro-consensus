# Results

Committed numerical results from the Phase 3 protocol modules, each with reproducibility metadata (parameters, RNG seeds, code version). This is a **Phase 3** deliverable; the directory is empty at Phase 0.

Per work-plan decision Q3, results are **committed as static files** (not generated on-demand) for citation stability — the Phase 3 gate requires documented results here, and on-demand-only generation would make that gate non-auditable.
