## Purpose

Coastline v0.4 registers its **principal open problem** (Deferred Items; Anti-Claim #6): the transplant of Zurek's redundancy / stability / compressibility criteria from *configurational* records (the quantum-Darwinism setting) to *temporal* records has no worked exemplar. The literature synthesis (`docs/literature/synthesis-v0.1.md`, §3.1) names three missing ingredients precisely:

1. a **temporal pointer basis** (what is einselected for a temporal record?);
2. a definition of an **independent temporal-record fragment**;
3. a system in which the *same* elapsed time is **redundantly** recorded across many carriers.

This note supplies a toy instrument that makes all three explicit and computes Claim II's **redundancy** leg as a number. It does so in the spirit of the Riedel–Zurek redundancy measure (Riedel & Zurek 2010; Zurek 2009): a partial-information plot and a redundancy count `R_δ`, here re-based on a *temporal* pointer.

What the note targets, concretely:

- A **definition** of a temporal-redundancy functional `R_δ[τ]` (§1).
- A **computation** distinguishing a *redundant* temporal record from an *anti-redundant* (entanglement-enhanced) one — making Coastline v0.4's "multipartite ≠ redundant" distinction numerical (§2).
- A **mapping** onto the motional-squeezing / Sorci case, so the functional concretises Ledger CL-2026-006's reading (§3).
- A **principled reason** for the resolution-anchor / classification-anchor split the roadmap defers into the Claim IV measure-registration protocol (§4).
- An **honest account** of what the functional does *not* settle, including that it does **not** close the principal open problem (§6).

## 1. The construction

### 1.1 Temporal pointer basis (ingredient i)

Fix a comparison protocol (Coastline v0.4 *Regime of Validity*) and a read-out epoch. Coarse-grain elapsed proper time into a finite **temporal pointer** `T` taking `d` values — the discrete labels of `d` resolvable elapsed-time bins, with phases `θ_j = 2π j / d` (`j = 0,…,d−1`) under a reference precession `ω`. The minimal non-trivial case is the **binary clock** `d = 2` (`T ∈ {early, late}`), used for all computations below; the generalisation to `d` bins is immediate.

`T` is *treated as* the pointer variable this instrument is about; its einselection is an assumed input, not derived here (§6). It is finite-dimensional by construction — this is deliberate and load-bearing (§4): redundancy is a statement about a *bounded* quantity of recorded information, `H_T ≤ log d`.

### 1.2 Independent temporal-record fragment (ingredient ii)

A **carrier** is any degree of freedom whose read-out yields a (possibly noisy) estimate of `T` (Coastline Claim I "temporal record"). A set of `N` carriers is **fragmentable** if, conditioned on `T`, the carrier read-outs are statistically independent — i.e. each carrier is an *independent* witness of the same elapsed-time bin. A **fragment** `F` is a subset of the `N` carriers; we write `f = m/N` for the fraction in a fragment of `m` carriers.

Conditional independence given `T` is exactly what distinguishes a *redundant* record (independent compatible copies) from an *entanglement-enhanced* one (jointly read, anti-redundant). It is the temporal analogue of the disjoint-photon-fragment decomposition in Riedel & Zurek (2010).

### 1.3 The functional (ingredient iii)

For a fragment of `m` carriers, define the **partial temporal information**

> `I(m) = I(T : F_m)`  — the mutual information between the temporal pointer `T` and the read-out of a fragment of `m` carriers.

A **redundant** temporal record is one whose partial-information plot `I(m)` rises to a **classical plateau** at `H_T` while `m ≪ N`: a small fragment already carries (almost) all the recordable elapsed-time information, and further carriers are redundant copies. The **temporal redundancy** at deficit `δ` is

> `R_δ[τ] = N / m_δ`,  where `m_δ = min { m : I(m) ≥ (1−δ) H_T }`.

`R_δ` is the *effective* number of disjoint fragments that each independently fix the elapsed-time bin to within deficit `δ` — the Riedel–Zurek `R_δ = 1/f_δ`, re-based on the temporal pointer of §1.1. It is a continuous ratio, not an integer tally: the literal count of disjoint fragments is `⌊N/m_δ⌋` (e.g. `⌊64/9⌋ = 7` for the §2 product record, whose ratio is `7.1`). A record whose plateau is reached only by the entire set (`m_δ = N`) has `R_δ = 1`: no redundancy. A record that never reaches `(1−δ)H_T` at all has `R_δ` **undefined** — a stronger failure, distinguished from `R_δ = 1` in §2.

## 2. Two analytic poles: redundant vs anti-redundant

We compute `I(m)` for the binary clock (`d = 2`, `H_T = 1` bit) in the two extreme configurations. In the **redundant** pole each carrier reads out a noisy copy of `T` with per-carrier flip probability `e ∈ (0, ½)` — the nuisance knob, standing for finite per-carrier distinguishability (dephasing, finite SNR). In the **anti-redundant** pole the global read-out is taken ideal (see below); `e` is therefore a property of the redundant pole only.

**Redundant pole (product record).** The `N` carriers are an unentangled product; each independently reads `T` with flip probability `e`. Conditioned on `T`, read-outs are i.i.d. — §1.2 holds maximally. This is the temporal analogue of Riedel & Zurek's scattered photons: independent carriers each imprinting the same bin.

**Anti-redundant pole (GHZ-shadow record).** The `N` carriers encode `T` so that only their *global* read-out carries it; every proper subset is marginally independent of `T`. The reference encoding is the cat state `(|0…0⟩ + (−1)^T |1…1⟩)/√2`, with the binary pointer stored in the relative phase (`0` or `π`): tracing out **any** carrier leaves the `T`-independent mixture `½|0…0⟩⟨0…0| + ½|1…1⟩⟨1…1|`, so every proper fragment carries *zero* elapsed-time information; only a coherent global (equatorial-parity) read-out of all `N` carriers recovers `T`. It is the canonical anti-redundant configuration — *many carriers, deliberately anti-redundant* (Coastline v0.4 Claim II terminology note; Claim V).

*Two cautions on this pole.* (i) The parity model used below is a **classical anti-redundancy model**: it reproduces the *information structure* (zero for every proper fragment, recoverable only from the whole) but does **not** derive its read-out statistics from the quantum measurement operators. (ii) The Heisenberg `F_τ ∝ N²` scaling often associated with such states belongs to the *continuous-phase* estimation problem of entanglement-enhanced metrology; it is invoked in §3–§4 as the physical motivation for "anti-redundant ⇒ high *joint* sensitivity," not as a property of this binary-pointer information count. To isolate the structural statement we take the global read-out as **ideal**, so the whole record attains `H_T`; any global read-out imperfection only lowers that value and never lifts the proper-fragment information off zero.

**Computed partial-information plot** (`N = 64`, `e = 0.20`; bits):

| `m` | `f = m/N` | `I(m)` — product | `I(m)` — GHZ-shadow |
| ---: | ---: | ---: | ---: |
| 1 | 0.016 | 0.2781 | 0.0000 |
| 2 | 0.031 | 0.4605 | 0.0000 |
| 4 | 0.062 | 0.6908 | 0.0000 |
| 8 | 0.125 | 0.8918 | 0.0000 |
| 16 | 0.250 | 0.9852 | 0.0000 |
| 32 | 0.500 | 0.9997 | 0.0000 |
| 48 | 0.750 | 1.0000 | 0.0000 |
| 64 | 1.000 | 1.0000 | 1.0000 |

The product record traces the characteristic concave rise to the `H_T = 1` bit plateau; `m_δ = 9` at `δ = 0.10`, so `R_{0.10} = 64/9 ≈ 7.1`. The GHZ-shadow record is a **step function**: `I(m) = 0` for every proper fragment, rising to `H_T = 1` bit only at `m = N` (ideal global read-out). Its redundancy follows from the *same* threshold definition — `m_δ = N`, hence `R_δ = N/N = 1`: the whole is the only fragment that records anything. (This is why the anti-redundant entry is `R_δ = 1`, not "undefined": unlike the noise-killed product rows below, the threshold *is* reached — but only by the entire set.)

This is the framework's "multipartite ≠ redundant" distinction as a number: the two configurations have the *same* `N` carriers and the GHZ pole carries *more* joint sensitivity in its continuous-phase realisation (§2 caution ii), yet `R_{0.10}` is `7.1` versus `1`.

**Redundancy vs per-carrier nuisance** (product pole, `N = 64`, `δ = 0.10`):

| per-carrier flip `e` | `m_δ` | `R_{0.10}` | `I(N)` (bits) |
| ---: | ---: | ---: | ---: |
| 0.10 | 4 | 16.0 | 1.000 |
| 0.20 | 9 | 7.1 | 1.000 |
| 0.30 | 22 | 2.9 | 0.998 |
| 0.40 | — | — | 0.807 |
| 0.45 | — | — | 0.354 |

As per-carrier distinguishability degrades, the plateau retreats and `R_δ` collapses; by `e ≳ 0.40` no fragment of the whole `N = 64` reaches the `90%` threshold and `R_{0.10}` is undefined. **A redundant record can be destroyed by per-carrier nuisance alone, without any change in entanglement structure** — the observation §5 turns on.

## 3. Motional squeezing and the Sorci case

The Sorci et al. (2026) protocol (Ledger CL-2026-006) is the minimal anti-redundant case: a clock degree of freedom `C` coupled to a *single* squeezed motional mode `M` through the mass-defect Hamiltonian

> `Ĥ = Ĥ_c + ℏω(n̂ + ½) − (ℏω / 2mc²) Ĥ_c P̂²`,

producing a Ramsey-visibility loss `V ≃ 1 − (ε_m ω_c t)² sinh²(2r) / 16` (CL-2026-006). In the functional's terms:

- The reduced clock `C`, obtained by tracing out `M`, is the natural **fragment** of the two-carrier `{C, M}` record. Its retained temporal information as a stand-alone record is bounded by its coherence `V`.
- Squeezing `r` raises the clock–motion entanglement, so `V` **falls** monotonically with `sinh²(2r)`. The clock-as-fragment therefore loses elapsed-time information as `r` grows: the **stability** leg (Claim II.2) fails under partial trace exactly as CL-2026-006 records, and the redundancy of the reduced clock falls.
- Meanwhile the *joint* `{C, M}` Fisher information about elapsed time **rises** with `r` — that is the metrological point of squeezing.

So on the `(R_δ, F_τ)` plane, increasing `r` moves the Sorci protocol *toward the GHZ pole*: redundancy down, joint Fisher up. This is the two-carrier (`N = 2`, "two carriers, not many") image of §2's `N`-carrier contrast, and it makes CL-2026-006's structural reading **concrete and directional** — visibility loss is a redundancy/stability failure bought by anti-redundant entanglement, not a gain in consensus. It does *not* yet supply a quantitative `R_δ(r)` curve: at `N = 2` the functional can only take the values `R_δ ∈ {1, 2}` (or be undefined), far too coarse to resolve a continuous dependence on `r`. A redundant trapped-ion temporal record would require the opposite design: *many independent, unentangled* carriers each reading the same elapsed-time bin (§2 product pole), which squeezing-based protocols deliberately do not build.

## 4. Why the functional is the classification anchor and Fisher is the resolution anchor

The roadmap defers, into the Claim IV measure-registration protocol, a distinction between a **resolution anchor** (bounds what must be resolved) and a **classification anchor** (carries the structural content classified), and logs that conflating them was the CL-2026-007 v0.1 failure mode. This functional supplies a *principled* reason for the split:

- **Fisher information is extensive.** For independent carriers `F_τ(m) = Σ_{k≤m} F_k = m·F_1` (product pole) — additive, **unbounded**, and **monotone increasing** in fragment size with no plateau. It measures *how finely* elapsed time can be resolved (Cramér–Rao), so it is the natural **resolution anchor**, but it cannot express redundancy: more carriers always means more Fisher, whether or not any single fragment already fixes the bin.
- **The mutual-information functional is bounded.** `I(m) ≤ H_T = log d` saturates, and the *fragment size at which it saturates* is precisely the redundancy statement. It is therefore the natural **classification anchor** for Claim II.1.

The GHZ pole shows the two anchors can move in **opposite directions**: in its continuous-phase realisation the joint Fisher information is maximal (Heisenberg `∝ N²`) while `R_δ = 1`. Registering Fisher as a redundancy measure would read this anti-redundant record as maximally redundant — exactly the inversion CL-2026-007 v0.1 fell into. The bounded-vs-extensive contrast is the structural reason the measure-registration protocol must register *one of each* and forbid substitution.

## 5. Bearing on the discriminant conditions (CL-2026-006 D1–D3)

The functional does **not**, by itself, discriminate structural consensus failure from nuisance — and seeing *why* sharpens the Ledger's discriminability gap. Two distinct causes lower `R_δ`:

- **Structural failure** (the witnessed effect): entanglement growth, tuned by squeezing `r` (and by the coupling `ε_m`), driving the record toward the anti-redundant pole.
- **Nuisance**: motional heating `ṅ`, dephasing `γ_φ`, preparation infidelity — all raise the per-carrier flip `e` and collapse the plateau (§2, second table), *with no change in entanglement structure*.

Both push `R_δ` down. The **value** of `R_δ` is therefore not a witness. What discriminates is its **controlled scaling**, which is exactly what CL-2026-006's discriminant conditions already demand, now expressed through the functional:

- **D3 (r-scan):** the structural contribution to the *information deficit* (via `V`) scales as `sinh²(2r)`; squeezing-preparation infidelity scales differently. Because `R_δ` is a *threshold* functional its own response is step-like, inheriting this `sinh²(2r)` dependence as a smooth curve only in the `N ≫ 2` limit; at the Sorci `N = 2` the observable is the structural *signature* (redundancy ↓ as joint Fisher ↑), not a fitted `R_δ(r)`.
- **D2 (null test):** the structural contribution tracks the coupling `ε_m` (lower trap frequency, heavier ion, lower clock frequency); the nuisance budget does not track `ε_m` the same way. `R_δ(ε_m)` in a null configuration bounds the nuisance floor that must be subtracted.
- **D1 (decomposition):** the per-carrier `e` is itself a sum of heating, dephasing, and detection contributions; the functional inherits whatever nuisance decomposition D1 supplies, and provides the redundancy observable that decomposition must explain.

In short: the functional concretises *where* the discrimination lives — in the controlled `r`- and `ε_m`-dependence, not in the bare value of `R_δ`. This is a candidate *measure-suggestive* sharpening of CL-2026-006's D2/D3; turning it into a measure-explicit `R_δ(r)` curve needs an `N ≫ 2` realisation (flagged for a future Ledger touch in §7).

## 6. What this note does and does not settle

**Does (modestly):**
- Gives the corpus's first *computed* partial-information plot and redundancy count `R_δ` re-based on a **temporal** pointer, with the three synthesis ingredients (§1) made explicit.
- Renders "multipartite ≠ redundant" numerical (§2) and supplies a structural reason for the resolution/classification-anchor split (§4).
- Maps both onto CL-2026-006 (§3) and onto its discriminant conditions (§5).

**Does not:**
- **It does not close the principal open problem, and does not touch Anti-Claim #6.** The product pole is a *candidate toy exemplar* of a redundant temporal record, not the demonstration the Coastline awaits. Four idealisations stand between it and that claim: (a) the temporal pointer is a coarse `d`-bin (here binary) label, not a continuous proper time; (b) the per-carrier read-out that turns each carrier's phase into a quasi-classical bit is *assumed* (an einselection input), not derived from a decoherence model; (c) the two poles are analytic endpoints — a non-toy instance would require a continuous-`τ`, dynamically einselected, many-carrier model (the natural next instrument: a master-equation computation of `R_δ` for `N` independently dephasing ions); (d) the anti-redundant pole is a *classical* parity model of the cat encoding `(|0…0⟩ + (−1)^T|1…1⟩)/√2`, reproducing its information structure (zero per proper fragment) but not derived from the quantum measurement operators (§2, caution i). Whether any such instance counts as *the* worked exemplar that converts Claim II from analogy to demonstration is a **Coastline-level judgement reserved to the steward**; this note, by Lock-Key discipline, makes no such edit.
- It does not classify any external claim (that is a Ledger function) and does not commit a Claim IV canonical measure (the Coastline declines to, at v0.4).

## Appendix A — Reproducibility (MIT)

The tables in §2 are regenerated by the following script (Python 3, standard library only). This code block is licensed **MIT** (it is code, per the repository split licence), independently of the CC BY-SA 4.0 note text.

```python
from math import comb, log2

def h2(p):
    if p <= 0 or p >= 1: return 0.0
    return -p*log2(p) - (1-p)*log2(1-p)

def I_product(m, e):
    """Mutual information I(T;F_m) in bits, binary pointer, m i.i.d. noisy copies."""
    H_T_given_F = 0.0
    for k in range(m+1):                       # k = number of carriers reading '1'
        p0 = comb(m, k) * e**k * (1-e)**(m-k)  # P(k | T=0)
        p1 = comb(m, k) * (1-e)**k * e**(m-k)  # P(k | T=1)
        pk = 0.5*p0 + 0.5*p1
        if pk > 0:
            H_T_given_F += pk * h2(0.5*p1/pk)
    return 1.0 - H_T_given_F                    # H_T = 1 bit

def I_ghz(m, N):                                # global-parity (GHZ-shadow) record;
    return 1.0 if m == N else 0.0               # ideal global read-out: I(N) = H_T

def redundancy(N, e, delta):
    for m in range(1, N+1):
        if I_product(m, e) >= (1-delta):
            return N/m, m
    return None, None

if __name__ == "__main__":
    N = 64
    for e in (0.10, 0.20, 0.30, 0.40, 0.45):
        R, m = redundancy(N, e, 0.10)
        print(f"e={e:.2f}  R_0.10={R if R else 'n/a'}  m_delta={m}  I(N)={I_product(N,e):.4f}")
    for m in (1, 2, 4, 8, 16, 32, 48, 64):
        print(f"f={m/N:.3f}  I_prod={I_product(m,0.20):.4f}  I_ghz={I_ghz(m,N):.4f}")
```
